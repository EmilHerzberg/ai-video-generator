LINKEDIN VISUAL DESIGN SYSTEM
“TECHNICAL OPERATOR”

CORE AESTHETIC

* Dark-mode first
* Minimal but layered
* Technical / analytical / strategic
* Futuristic without sci-fi cliché
* Looks like internal tooling from a top AI startup
* Clean information density
* Motion-ready by default

FORMAT

* 1080x1350 portrait (best LinkedIn engagement)
  Secondary:
* 1920x1080 landscape for videos
* 1080x1080 for diagrams/charts

* #0A0F1C

VISUAL STRUCTURE

“a screenshot from a strategic intelligence system.”

Use:

* Flowcharts

* System diagrams

* Huge emoji overload

PIPELINE SUGGESTION

BEST SIMPLE PIPELINE:

1. CONTENT GENERATION
   Claude / GPT
   ↓
   Generate:

* headline
* structure
* chart ideas
* animation metadata

2. VISUAL TEMPLATE ENGINE
   React + Tailwind + Framer Motion

Build reusable components:

* charts
* cards
* flowcharts
* node systems
* timelines
* terminal windows

3. VIDEO RENDERING
   Use Remotion

Reason:

* programmatic video generation
* React-based
* scalable
* perfect for reusable templates
* motion graphics from code

4. AUTOMATION
   Claude CLI generates JSON config:

Example:
{
"title": "AI Agents Will Replace SaaS Layers",
"nodes": [...],
"charts": [...],
"highlights": [...]
}

Then your React/Remotion system renders automatically.

5. OUTPUT
   Auto-render:

* PNG carousel
* MP4 short video
* GIF snippets
